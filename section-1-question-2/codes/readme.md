## Overview

This repository contains a Jupyter Notebook (.ipynb) that performs the full analysis, including data preparation, modeling, and result generation.

## How to Run

Open the notebook in Jupyter Notebook or JupyterLab.

## Run the cells from top to bottom.

No additional setup or configuration is required beyond the standard Python environment.

## Data

All datasets required for the analysis are located in the data/ folder.

The notebook reads directly from this folder; no manual data loading is needed.

## Notes

All required data processing and analysis steps are contained within the notebook.

Outputs (figures, tables, and results) are generated automatically when the notebook is run.
